# 问卷星自动化填写脚本

## 简介

这是一个用于自动填写问卷星问卷的Python脚本，使用Selenium进行网页自动化操作，支持多种题型的自动填写，包括单选题、多选题、填空题、矩阵题、量表题等。

## 功能特性

- 支持多种题型：单选题、多选题、填空题、矩阵题、量表题、下拉框题、排序题、滑块题
- 可配置选项概率：可以根据设定的概率分布自动选择答案
- 多线程支持：可并发填写多份问卷，提高效率
- IP代理支持：可配置代理IP避免被限制
- 自动提交：自动处理问卷提交和验证过程

## 环境要求

- Python 3.x
- Chrome浏览器
- ChromeDriver（需与Chrome版本匹配）
- 以下Python包：
  - selenium
  - numpy
  - requests

## 配置说明

### 1. 浏览器路径配置
chrome浏览器驱动及chrome浏览器下载地址：https://developer.chrome.com/docs/chromedriver/downloads?hl=zh-tw
注：下载时需要保证版本一致，如已经下载版本浏览器，可在chrome浏览器打开chrome://version 查看对应版本号，再下载对应版本的chromedrive

```python
option.binary_location = r"chrome.exe路径"
```

### 2. 问卷链接配置

修改 `url` 变量为您要填写的问卷链接：

```python
url = "https://v.wjx.cn/vm/e72K0lp.aspx"  # 替换为您的问卷链接
```

### 3. 题目参数配置

#### 单选题概率参数 (`single_prob`)
- 键：题目编号（仅用于标识，不严格对应实际题号）
- 值：选项概率数组，数组长度必须与题目选项数量一致
- 例如 `[1, 1, 1]` 表示三个选项被选中的概率相等
- 使用 `-1` 表示完全随机选择

```python
single_prob = {
    "1": [1, 1],           # 第1题有2个选项
    "2": [1, 1, 1],        # 第2题有3个选项
    "3": [5, 17.3, 13.6, 25.7, 8.4],  # 第3题有5个选项
    # ...
}
```

#### 多选题概率参数 (`multiple_prob`)
- 数值表示每个选项被选中的百分比概率
- 例如 `[20, 20, 20, 20, 20]` 表示每个选项有20%的概率被选中

#### 矩阵题概率参数 (`matrix_prob`)
- 与单选题类似，但应用于矩阵题的每一行

#### 量表题概率参数 (`scale_prob`)
- 适用于量表题（如1-5分评分）

#### 填空题内容 (`texts`)
- 为填空题提供预设的答案选项

### 4. 目标份数和线程数配置

在 `main` 函数中修改：

```python
target_num = 3        # 目标填写份数
num_threads = 2       # 并发线程数（浏览器窗口数）
```

## 重要注意事项

### 参数配置准确性
- 每道题的概率数组长度必须与实际选项数量完全一致
- 不要配置不存在的题目参数，会导致 `IndexError: list index out of range` 错误
- 仔细核对每道题的选项数量与配置参数

### 常见错误及解决方案

1. **参数长度不匹配错误**：
   ```
   "第X题参数长度：Y,选项长度Z,不一致！"
   ```
   - 解决方案：检查并修正该题的概率数组长度，使其与实际选项数匹配

2. **索引超出范围错误**：
   ```
   IndexError: list index out of range
   ```
   - 解决方案：删除多余的概率参数配置，确保参数数量不超过实际题目数量

3. **浏览器路径错误**：
   - 确认Chrome和ChromeDriver路径配置正确

## 高级功能

### 失败重试机制
